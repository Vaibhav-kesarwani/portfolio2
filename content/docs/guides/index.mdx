# Welcome to Lab's

**Lab's** is a modern component library that brings together the best of React, Tailwind CSS,
and Framer Motion to create beautiful, accessible, and performant user interfaces.
Each component is carefully crafted with modern animations and thoughtful design principles.

## Key Features

- **shadcn CLI Compatible**: Install components using the familiar shadcn CLI
- **TypeScript First**: Full type safety with comprehensive TypeScript support
- **Accessible by Default**: Built with accessibility best practices
- **Dark Mode Support**: All components work seamlessly in light and dark themes
- **Customizable**: Easy to customize with Tailwind CSS classes
- **Performance Optimized**: Built with performance in mind using modern React patterns

## Frequently Asked Questions

<Accordions>
  <Accordion id="is-labs-free" title="Is Lab's free to use?">
    Yes! Lab's is completely free and open source. You can use it in personal
    and commercial projects.
  </Accordion>
  <Accordion
    id="do-i-need-to-know-framer-motion"
    title="Do I need to know Framer Motion to use Lab's Compoents?"
  >
    No! All animations are built-in. You can use components without any Framer
    Motion knowledge.
  </Accordion>
  <Accordion
    id="can-i-customize-the-components"
    title="Can I customize the components?"
  >
    Absolutely! All components are built with Tailwind CSS and can be customized
    using standard Tailwind classes.
  </Accordion>
  <Accordion
    id="does-components-work-with-nextjs"

  </Accordion>
</Accordions>
